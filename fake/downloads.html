Updating

  1. open Lantern
  2. navigate to Account
  3. select Settings
  4. choose Check for Updates

If an update cannot be completed in the application, follow the Downloads instructions to update.

Check Version

  1. open Lantern
  2. navigate to Account
  3. select Settings
  4. version number is listed on bottom of the screen

Check our release notes to see which version is newest, and always ensure you’re updated to the latest version to access the latest protocols and most effective proxies.
